## Why this repository?

Most AI courses assume you already know how to code, or jump straight into frameworks without explaining *why* things work. This repo is different:

- **Self-contained notes** — each topic is a complete lesson you can read top-to-bottom, no video required.
- **Zero-to-RAG path** — we start with `print("Hello World")` and build up to vector databases and RAG.
- **Runnable code** — every concept comes with small, focused Python examples.
- **Concept-first** — real-world analogies before syntax, so the ideas actually stick.

> If you find these notes useful, **star the repo** — it helps others discover them too.

---

## Who is this for?

- Beginners who want to learn **Python from scratch** with an AI end-goal.
- Developers moving into **Generative AI and Agentic systems**.
- Anyone who wants to understand **LLMs, embeddings, vector search, and RAG** deeply — not just call an API.

## What you'll learn

By the end of this course you'll be able to:

1. Write Python confidently — data types, control flow, functions, and OOP.
2. Use Git & GitHub for version control.
3. Explain what an **AI agent** is and design one from its core components.
4. Apply **prompt engineering** and self-reflection / feedback-loop patterns.
5. Give agents **memory** (short-term & long-term).
6. Store and query data with **SQL** and **vector databases**.
7. Build a working **RAG pipeline** from ingestion to answer generation.
